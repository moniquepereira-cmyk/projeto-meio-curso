Bob Esponja Burguer & Restaurante

Este projeto é um site institucional inspirado em um restaurante temático do Bob Esponja em São Paulo. O objetivo é apresentar informações sobre o local, cardápio, experiência e contato de forma organizada e visual.

O site foi desenvolvido utilizando HTML5 e CSS3, com estrutura semântica e organização modular do CSS. A estilização foi dividida em dois arquivos: base.css, responsável pelas configurações globais como reset, tipografia e variáveis de cores; e layout.css, responsável pelo cabeçalho, menu com Flexbox, rodapé e organização visual das páginas.

O projeto possui quatro páginas principais: página inicial, sobre o restaurante, cardápio e contato com endereço e horários.

A identidade visual utiliza cores inspiradas no tema do Bob Esponja, com destaque para azul e amarelo, mantendo consistência em todas as páginas.

Este projeto foi desenvolvido para fins acadêmicos.
